# CLOCK

Sets the date and time on the Master.

The date and time settings are propagated over the local bus.

Syntax:

```
'CLOCK \<mm-dd-yy\> \<hh:mm:ss\>'

```
Example:

```
SEND_COMMAND 0,"'CLOCK 04-12-05 09:45:31'"

```
