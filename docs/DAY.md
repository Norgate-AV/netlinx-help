# DAY

This system variable returns the current day of the week as one of the following strings:

'MON', 'TUE', 'WED', 'THU', 'FRI', 'SAT' or 'SUN'.

Example:

```c linenums="1"
IF (DAY = 'SUN')

{

}

```
