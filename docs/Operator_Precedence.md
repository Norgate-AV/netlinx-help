# Operator Precedence

The table below shows the inherent precedence assigned to the Operators.

|       |                      |               |
|-------|----------------------|---------------|
| Level | Operators            | Associativity |
| 1     | ! ~                  | Left To Right |
| 2     | \* / %               | Left To Right |
| 3     | \<\< \>\>            | Left To Right |
| 4     | \+ -                 | Left To Right |
| 5     | \< \<= \> \>= = \<\> | Left To Right |
| 6     | & \| ^               | Left To Right |
| 7     | && \|\| ^^           | Left To Right |
