# DEFINE_TYPE

The DEFINE_TYPE section is used to define custom data types such as structures
and classes.

Example:

```
DEFINE_TYPE

```

 

STRUCTURE MyStruct

{

 LONG Num

 CHAR Name\[30\]

}
