# LOG_VALUE

Note: Math functions are supported by NetLinx Controller firmware version 4 or
higher.

Returns the natural logarithm of x.  The natural logarithm is the base-e
logarithm, the inverse of the natural exponential function (EXP_VALUE) .

LOG_VALUE(CONSTANT VARIANT X)

Where X can be any intrinsic type (INTEGER, FLOAT,  DOUBLE, etc) with a value
greater than 0.
