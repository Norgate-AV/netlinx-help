# Duplicate symbol

Duplicate definitions of [variables](Variables.md) or [constants](Constants.md) are found.

All variables and constants must have unique identifiers.

Note: Unlike Compiler Warnings, Compiler Errors must be corrected before your NetLinx program can be executed.
