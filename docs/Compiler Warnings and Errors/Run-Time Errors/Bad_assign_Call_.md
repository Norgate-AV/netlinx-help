# Bad assign Call...

These errors occur if the Central Controller cannot assign a parameter in a CALL
statement to the parameter in the corresponding DEFINE_CALL statement.

Note: Unlike Compiler Warnings, Compiler Errors must be corrected before your
NetLinx program can be executed.
