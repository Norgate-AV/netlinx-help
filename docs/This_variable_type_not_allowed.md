# This variable type not allowed

This error occurs when an attempt is made to use an array variable with [DEFINE\_LATCHING](DEFINE_LATCHING.md) , [DEFINE_TOGGLING](DEFINE_TOGGLING.md), or [DEFINE_MUTUALLY_EXCLUSIVE](DEFINE_MUTUALLY_EXCLUSIVE.md).

Note: Unlike Compiler Warnings, Compiler Errors must be corrected before your NetLinx program can be executed.
