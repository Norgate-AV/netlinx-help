# DYNAMIC_APPLICATION_DEVICE

Specifies a Duet device that is completely dynamic.  

A dynamically discovered device matching the specified deviceType could be bound to the duetDevice from anywhere in the system.

Syntax:

```
DYNAMIC_APPLICATION_DEVICE (DEV duetDevice,  char\[\] deviceType,  char\[\] friendlyName)

```
