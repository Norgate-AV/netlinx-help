# DEV

A data type (structure) used to represent a specific device, port and system.

In NetLinx, the DEV structure is the actual (internal) representation of a NetLinx device.

Example:

```c linenums="1"
STRUCTURE DEV

{

```
INTEGER Number

INTEGER Port

INTEGER System

}
