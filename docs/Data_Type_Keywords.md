# Data Type Keywords

NetLinx supports integers up to 32-bits and signed values to allow positive and negative values.

The NetLinx language supports the following Date Type keywords:

Intrinsic:

- [CHAR](CHAR.md)

- [DOUBLE](DOUBLE.md)

- [FLOAT](FLOAT.md)

- [INTEGER](INTEGER.md)

- [LONG](LONG.md)

- [SINTEGER](SINTEGER.md)

- [SLONG](SLONG.md)

- [WIDECHAR](WIDECHAR.md)

Structured:

- [DEV](DEV.md)

- [DEVCHAN](DEVCHAN.md)

- [DEVLEV](DEVLEV.md)
