# PAUSE_WAIT

This command suspends the specified (named) [WAIT](WAIT.md) until a [RESTART_WAIT](RESTART_WAIT.md), [RESTART_ALL_WAIT](RESTART_ALL_WAIT.md), [CANCEL_WAIT](CANCEL_WAIT.md) or [CANCEL_ALL_WAIT](CANCEL_ALL_WAIT.md) command is issued.

Syntax:

```c linenums="1"
PAUSE_WAIT \<'name'\>

```
