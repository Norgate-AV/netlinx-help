# DEVLEV

A data type (structure) containing fields used to represent a specific device number, port, system and level.

This structure is used to implement an array DEVLEV\[ \].

Example:

```c linenums="1"
STRUCTURE DEVLEV

{

```
 DEV  Device

 INTEGER Level

}
