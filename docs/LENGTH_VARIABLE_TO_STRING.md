# LENGTH_VARIABLE_TO_STRING

This routine calculates how many bytes it takes to encode a variable.

Parameters:

- Encode - any type of variable. This is the variable to be encoded.

Result:

- Number of bytes required to encode variable.

- 0 Encoded variable error, unrecognized type
