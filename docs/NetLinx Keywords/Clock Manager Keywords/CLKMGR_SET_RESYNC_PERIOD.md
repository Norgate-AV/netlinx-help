# CLKMGR_SET_RESYNC_PERIOD

Sets the re-sync period to the specified minute value.

Syntax:

```
CLKMGR_SET_RESYNC_PERIOD (CONSTANT INTEGER PERIOD)

```
The upper bound is 480 minutes (i.e., 8 hours).
