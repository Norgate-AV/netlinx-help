# CLKMGR_SET_ACTIVE_TIMESERVER

Sets the time server entry that has the matching IP-ADDRESS to the IP parameter as the active time server entry.

Syntax:

```
CLKMGR_SET_ACTIVE_TIMESERVER (CONSTANT CHAR IP\[\])

```
