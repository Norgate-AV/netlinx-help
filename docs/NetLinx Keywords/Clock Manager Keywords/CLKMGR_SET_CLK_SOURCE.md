# CLKMGR_SET_CLK_SOURCE

Sets the source of the clock manager.

Syntax:

```
CLKMGR_SET_CLK_SOURCE (CONSTANT INTEGER MODE)

```
Can be set to:

 CLKMGR_MODE_NETWORK

or

 CLKMGR_MODE_STANDALONE
