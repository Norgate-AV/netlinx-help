# GET_LOG_LEVEL

Retrieves the current log level for the program.  

INTEGER GET_LOG_LEVEL()

Where the returned value will be one of the for log levels:

- INTEGER  AMX_ERROR = 1

- INTEGER  AMX_WARNING = 2

- INTEGER  AMX_INFO = 3

- INTEGER  AMX_DEBUG = 4
