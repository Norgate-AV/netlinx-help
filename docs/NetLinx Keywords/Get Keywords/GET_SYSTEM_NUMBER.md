# GET_SYSTEM_NUMBER

Returns the system number of the NetLinx Master.

Syntax:

```
INTEGER GET_SYSTEM_NUMBER( )

```
Parameters:

None

Note: Â When it is a NetLinx function the ( ) are NOT OPTIONAL - even if there are no parameters.

Result:

An integer representing the system number of the NetLinx Master.

Example:

```
SystemNum = GET_SYSTEM_NUMBER() // get local system num

```
