# CLKMGR_DELETE_USERDEFINED_TIMESERVER

Deletes the user-defined entry that has its IP-ADDRESS matching the parameter.

Syntax:

```
CLKMGR_DELETE_USERDEFINED_TIMESERVER (CONSTANT CHAR IP\[\])

```
