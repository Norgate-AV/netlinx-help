# RELEASE_CHANNEL not allowed within Wait

The [RELEASE_CHANNEL](RELEASE_CHANNEL.md) keyword is not allowed in a section of code which will be executed due to a [WAIT](WAIT.md) keyword.

Note: Unlike Compiler Warnings, Compiler Errors must be corrected before your NetLinx program can be executed.
