# CANCEL_ALL_WAIT_UNTIL

This keyword cancels all (named or unnamed) [WAIT_UNTIL](WAIT_UNTIL.md) and [TIMED_WAIT_UNTIL](TIMED_WAIT_UNTIL.md) commands from the appropriate wait list.

Syntax:

```
CANCEL_ALL_WAIT_UNTIL \<'name'\>

```
