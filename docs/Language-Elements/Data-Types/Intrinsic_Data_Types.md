# Intrinsic Data Types

The following Data Types are intrinsic to the NetLinx language:

| Keyword                 | Data Type                       | Sign     | Size   | Range                                |
| ----------------------- | ------------------------------- | -------- | ------ | ------------------------------------ |
| [CHAR](CHAR.md)         | Byte                            | Unsigned | 8-bit  | 0 -255                               |
| [WIDECHAR](WIDECHAR.md) | Integer                         | Unsigned | 16-bit | 0 - 65535                            |
| [INTEGER](INTEGER.md)   | Integer                         | Unsigned | 16-bit | 0-65536                              |
| [SINTEGER](SINTEGER.md) | Integer                         | Signed   | 16-bit | -32768 to +32768                     |
| [LONG](LONG.md)         | Long Integer                    | Unsigned | 32-bit | 0 - 4294967295                       |
| [SLONG](SLONG.md)       | Long Integer                    | Signed   | 32-bit | + 2,147,483,647                      |
| [FLOAT](FLOAT.md)       | Floating Point                  | Signed   | 32-bit | 1.79769313 E+308 to 2.22507385 E-308 |
| [DOUBLE](DOUBLE.md)     | Double Precision Floating Point | Signed   | 64-bit | 3.40282347 E+38 to 1.17549435 E-38   |
