# Log Keywords

The NetLinx programming language supports the following Log keywords:

- [AMX_LOG](AMX_LOG.md)
- [SET_LOG_LEVEL](SET_LOG_LEVEL.md)
- [GET_LOG_LEVEL](GET_LOG_LEVEL.md)

Where LOG_LEVEL is one of the following built-in constants:

- INTEGER  AMX_ERROR = 1;
- INTEGER  AMX_WARNING = 2;
- INTEGER  AMX_INFO = 3;
- INTEGER  AMX_DEBUG = 4;
