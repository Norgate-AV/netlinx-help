# Device values must be equal

In a range specification, the devices (or their defined identifiers) must be
equal.

For example, (\[1,1\]..\[1,5\] ) is valid; (\[1,1\]..\[2,5\] ) is not.

Note: Unlike Compiler Warnings, Compiler Errors must be corrected before your
NetLinx program can be executed.
