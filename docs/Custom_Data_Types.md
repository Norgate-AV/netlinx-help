# Custom Data Types

Custom data types such as structures and classes are defined in the [DEFINE_TYPE](DEFINE_TYPE.md) section.

Example:

```c linenums="1"
DEFINE_TYPE

```
 

STRUCTURE MyStruct

{

 LONG Num

 CHAR Name\[30\]

}
